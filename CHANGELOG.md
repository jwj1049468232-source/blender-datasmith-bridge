## [Unreleased]

## [1.1.0] - 2026-04-23

### Added
- ✨ 新增 UE5 实时同步功能 - 导出后自动通知 UE5 重新导入
- 🎯 新增集合导出模式 - 自动根据集合名称命名文件
- 📂 新增快速打开输出目录功能
- 🔧 新增导出选项：应用修改器、导出动画
- 🌐 新增跨平台支持（Windows/macOS/Linux）

### Changed
- 📝 完善作者信息和版权说明
- 📄 添加 MIT 许可证
- 🎨 优化 UI 界面，增加图标和分组

### Fixed
- 🐛 修复集合名称包含空格时的匹配问题
- 🐛 修复相机视图删除时的崩溃问题

## [1.0.0] - 2026-04-01

### Added
- 🎉 初始版本发布
- 📤 基础 Datasmith 导出功能
- 📁 支持自定义输出路径
- ✅ 支持仅导出选中对象
